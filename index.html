Latest Project

ChefRobot

This innovative web application harnesses the power of Azure OpenAI API and llms models to create recipes interacting with a chatbot just providing the name of the recipe you want to cook. The chatbot will provide you with the ingredients , steps to follow to cook the recipe using gpt3.5 model and an image of the recipe using the DALL.E 3 model . In this project i'v used Spring AI and Spring Boot to create the chatbot and the backend of the application and vanillaJs(no frontend framework) , css and HTML to create the frontend of the app
